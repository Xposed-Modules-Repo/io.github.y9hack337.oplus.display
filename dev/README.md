# Developer Docs

This folder contains only reverse-engineering docs for:

- pulling APKs from device
- decompiling DEX
- parsing output and UI dumps

## Files

- `RECOVERY_PLAYBOOK.md`
  Minimal troubleshooting flow focused on extraction/decompilation/parsing.
- `REVERSE_ENGINEERING_SYSTEMUI.md`
  Full SystemUI APK reverse-engineering commands.
- `APK_DECOMPILE_AND_PARSE_CHECKLIST.md`
  Decompilation/parsing-only checklist.
- `scripts/parse_uiautomator_dump.py`
  XML parser for `uiautomator` dumps.
